# Калькулятор логистики труб

Веб-приложение для расчёта оптимальной загрузки стальных труб в транспорт (фура / полувагон) с 2D-визуализацией укладки.

## 🚀 Возможности

- **2D-симуляция укладки труб** — точный расчёт размещения с учётом габаритов
- **Оптимизация смешанных загрузок** — минимизация количества транспорта
- **Визуализация сечений** — вид спереди и сбоку для каждого авто
- **Интеграция с 1С** — загрузка заявок в формате JSON/TXT
- **Нормы Уралтрубпром** — для полувагонов (Ø114–630 мм)
- **10 материалов** — сталь, алюминий, медь, титан и др.

## 📦 Установка

Проект не требует установки зависимостей — работает из одного HTML-файла:

```bash
git clone https://github.com/karpvch-cpu/pipe-logistics-calc.git
cd pipe-logistics-calc
```

Откройте `index.html` в браузере или разверните на любом веб-сервере.

## 📝 Использование

### 1. Ручной ввод

1. Выберите тип транспорта (Фура / Полувагон)
2. Укажите материал (по умолчанию: сталь)
3. Добавьте позиции труб: диаметр, толщина, длина, количество
4. Нажмите **"РАССЧИТАТЬ С ОПТИМИЗАЦИЕЙ"**

### 2. Загрузка из 1С

Подготовьте JSON-файл в формате:

```json
{
  "requestId": "ЗК-000001247",
  "client": "ООО Компания",
  "material": "steel",
  "positions": [
    {
      "diameter": 219,
      "thickness": 6,
      "length": 12000,
      "count": 117,
      "weightPerMeter": 31.52,
      "totalMeters": 1404
    }
  ]
}
```

Нажмите **"📁 ЗАГРУЗИТЬ ФАЙЛ"** и выберите JSON или TXT.

## 🎯 Особенности расчёта

### Габариты транспорта

**Фура:**
- Длина: 13.6 м
- Ширина: 2.44 м
- Высота: 2.58 м
- Макс. вес: 22 тонны

**Полувагон:**
- Длина: 12.68 м
- Ширина: 2.83 м
- Высота: 2.17 м
- Макс. вес: 63 тонны

### Алгоритм укладки

1. **Сортировка по весу** — тяжёлые трубы вниз, лёгкие наверх
2. **Комбинаторная оптимизация** — перебор всех комбинаций (до 8 позиций)
3. **Жадные стратегии** — 4 эвристики для больших заявок
4. **2D-симуляция** — точное размещение по ширине и высоте
5. **Проверка по весу** — ограничение грузоподъёмности

### Формула веса погонного метра

```
W (кг/м) = π × t × (D − t) / 1 000 000 × ρ
```

Где:
- **D** — наружный диаметр (мм)
- **t** — толщина стенки (мм)
- **ρ** — плотность материала (кг/м³)

## 📐 Нормы загрузки

Для полувагонов используются нормы Уралтрубпром:

| Диаметр | Россыпью | Пакеты станд. | Пакеты выс. |
|---------|----------|---------------|-------------|
| Ø114    | 361      | 325           | 361         |
| Ø219    | 117      | 90            | 110         |
| Ø426    | 30       | 30            | 36          |
| Ø630    | —        | —             | 19 (с шапкой) |

Для остальных диаметров — геометрический расчёт.

## 📄 Формат обмена с 1С

Подробная спецификация в файле [`docs/ТЗ_обмен_1С.md`](docs/ТЗ_обмен_1С.md)

**Обязательные поля:**
- `diameter` — диаметр (мм)
- `thickness` — толщина стенки (мм)
- `length` — длина трубы (мм)
- `count` — количество (шт)

**Опциональные поля:**
- `weightPerMeter` — вес м/п (кг/м)
- `totalMeters` — общая длина (м)
- `material` — код материала

## 🎨 Визуализация

Для каждого авто отображается:

**Вид спереди (сечение с торца):**
- Круги труб с диаметрами
- Цветовая кодировка по диаметру
- Размеры кузова

**Вид сбоку (продольное сечение):**
- Слои труб разной высоты
- Масштаб 1:2 для компактности
