## 🎯 Назначение

Веб-калькулятор для расчёта загрузки стальных труб в транспортное средство.  
Работает как один standalone HTML-файл, не требует сборки или сервера.

## ⚙️ Ключевые возможности

### Укладка
- **Гравитационная симуляция** — каждая труба падает на самое низкое доступное место, опирается на пол или в «седло» между двумя соседними трубами
- **Гексагональная упаковка** для всех диаметров (коэффициент высоты яруса √3/2 ≈ 0.866)
- **Смешанная загрузка** — трубы разных диаметров заполняют пустоты друг между другом
- **Ряды по длине** — трубы 6 м размещаются в 2 ряда по длине фуры (13.6 м)

### Расчёт
- Формула веса погонного метра: `W = π × t × (D − t) / 1 000 000 × ρ`
- 10 материалов: сталь, нержавейка, медь, алюминий, латунь, титан, чугун, никель, цинк, свинец
- Оптимизация — минимизация количества транспортных средств
- Проверка габаритов: длина трубы > длины кузова → блокировка расчёта с уведомлением

### Загрузка данных
- **JSON** (формат 1С) — поле `positions` с массивом позиций
- **Excel** (.xlsx / .xls) — автоматический поиск колонок по ключевым словам на русском и английском
  - Диаметр, стенка, наименование, количество
  - Длина с автоопределением единиц: **м → ×1000**, **см → ×10**, **мм → без изменений**
- Извлечение диаметра и стенки из наименования по шаблону `114,3×6` или `219х10`
- Поддержка дробных диаметров: `114,3`, `323,9`, `406,4` и т.д.

### Визуализация
- **Вид спереди** — реальные координаты каждой трубы в сечении
- **Вид сбоку** — ряды по длине, свободное пространство, высота по слоям
- Цветовая кодировка по диаметрам

## 📁 Формат JSON (1С)

```json
{
  "requestId": "ЗК-000001247",
  "requestDate": "2026-02-18",
  "client": "ООО ТрубоСтрой",
  "material": "steel",
  "positions": [
    {
      "name": "Труба бесшовная 219х6 ГОСТ 8732-78",
      "diameter": 219,
      "thickness": 6,
      "length": 12000,
      "count": 100
    }
  ]
}
```

## 📊 Формат Excel

Калькулятор ищет колонки по ключевым словам (регистр не важен):

| Данные | Ключевые слова |
|--------|---------------|
| Наименование | наим, name, номенкл, описан |
| Диаметр | диам, diameter, dn, ∅ |
| Стенка | стенк, толщ, thickness, wall |
| Длина | длина одной, длин, length, len |
| Количество | кол, qty, count, колич, шт |

## 🚀 Деплой на GitHub Pages

1. Создать репозиторий на [github.com/new](https://github.com/new) (Public)
2. Загрузить `index.html` через **Add file → Upload files**
3. **Settings → Pages → Branch: main → Save**
4. Сайт доступен: `https://ВАШ_ЛОГИН.github.io/РЕПОЗИТОРИЙ/`
